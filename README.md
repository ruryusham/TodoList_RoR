●全体の設計・構成についての説明
本ToDoリストでは，
  ・ToDoリスト
  ・タスク(ToDo)
が登録できるアプリケーションになっている．
タスクは期限と実行有無を付加情報として持っており，削除が可能となっている．
また，ToDoリストに関しては名称を変更する事が可能である．

トップページとリストの詳細ページ，検索画面が主要なページになっており，
トップページではそれぞれのリストの中で直近の期日が表示されるようになっている．

・rbenvのインストール
・Ruby on Railsのインストール
